# 通用组件库

本项目包含多个可复用的 UI 组件，下文聚焦它们在界面上的呈现与交互表现。

## 1. ItemTag

用于在文本中或列表中展示物品，支持点击查看详情。

```text
+-----------------------+
| [Icon?] Item Name     |
+-----------------------+
```

- **交互**: 点击可打开物品详情弹窗，适合在文本或列表中嵌入。
- **视觉**: 圆角胶囊样式，浅色背景，悬停时轻微变色。

## 2. ActionQueue

当前行动指示器 + 队列管理入口。组件的详细规格（布局、状态、交互、Modal 等）已经拆分到 [ActionQueue & ActionQueueModal 规格](components/ACTION_QUEUE.md)。

- **用途概述**: 在主界面顶部展示正在执行的技能或战斗，同时提供停止/打开队列的入口。
- **核心交互**: Queue 按钮唤起队列弹窗；Stop 按钮立即清空当前行动。
- **视觉提示**: 采集动作显示单条进度条；战斗动作以双行 HP/MP 彩条呈现；空态仅显示 “Nothing”。
- **尺寸约束**: 固定 `max-width` 避免按钮显隐导致宽度跳动。
- **更多细节**: 包括空态、按钮显隐、Modal 排序/删除策略，请参见专门文档。

## 3. ModalBox

通用的模态框容器，支持堆叠管理、点击背景关闭、ESC 关闭。

```text
+-------------------------------------------------------+
|  Overlay (Black with opacity)                         |
|                                                       |
|  +-------------------------------------------------+  |
|  |  Modal Content                                  |  |
|  |                                                 |  |
|  |  [Header]                                       |  |
|  |                                                 |  |
|  |  ...                                            |  |
|  |                                                 |  |
|  +-------------------------------------------------+  |
|                                                       |
+-------------------------------------------------------+
```

- **交互**:
    - 点击背景或按下 ESC 可关闭当前模态框。
    - 聚焦在模态区域内循环移动，保证键盘可用性。
    - 多个模态并存时，仅最上层对键盘事件做出响应。

## 4. LeftSidebar

侧边栏导航组件，负责显示技能列表和路由跳转。

- **特点**:
    - 按“生产 / 战斗”逻辑聚合技能入口。
    - 桌面端展示技能名称、等级与进度；移动端仅保留简写标签。
    - 自带宝箱、战斗、背包等快捷入口。

## 5. NotificationCenter

全局通知中心，用于显示 Toast 消息（如升级提示、获得物品）。

```text
+---------------------------------------+
| [Info] Level Up!                      |
| You reached Mining level 10.          |
+---------------------------------------+
| [Success] Item Added                  |
| Copper Ore +1                         |
+---------------------------------------+
```

- **Position**: 固定在屏幕右下角 (Desktop) 或 底部 (Mobile)。
- **Behavior**: 消息会自动消失。

## 6. EnemyList

用于战斗页面，展示可攻击的敌人列表。

```text
+-------------------+  +-------------------+
| Rat (Lvl 1)       |  | Goblin (Lvl 3)    |
| HP: 50  Atk: 5    |  | HP: 120 Atk: 12   |
| Drops: Tail, Gold |  | Drops: Cloth      |
| [Select]          |  | [Select]          |
+-------------------+  +-------------------+
```

- **交互**: 卡片支持点击选择敌人并打开战斗配置弹窗。
- **展示内容**: 名称、等级、基本属性、掉落摘要及操作按钮。

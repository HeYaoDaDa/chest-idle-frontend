# ActionQueue 主组件

在所有主要页面中固定显示当前执行行动，并提供队列管理与停止入口。

---

## 空态

```
┌──────────────────────────────────────────────────┐
│ Nothing                                          │
└──────────────────────────────────────────────────┘
```

仅显示"Nothing"文案，隐藏控制区和进度区。

## 生产行动运行中

```
┌──────────────────────────────────────────────────┐
│ Mining · Copper Ore · ∞            [Queue] [Stop]│
├──────────────────────────────────────────────────┤
│ [███████████████████ 1.2s ────────────────────-] │
└──────────────────────────────────────────────────┘
```

- 信息区：`Skill Name · Action Name · Amount`，其中 Skill Name 为技能名（战斗时显示"战斗"），Action Name 为技能/资源名称，Amount 为整数或 `∞`
- 控制区：Queue 按钮显示 `Queue (n)`（n 为待执行行动数量），Stop 按钮固定显示；按钮宽度固定避免抖动
- 进度区：单条灰底圆角进度条，蓝色填充，中央显示单次耗时（如 `1.2s`）；高度 12px，300ms 平滑过渡
- Queue 按钮行为：点击打开 ActionQueueModal
- Stop 按钮行为：立即停止当前行动

## 战斗行动运行中

```
┌──────────────────────────────────────────────────┐
│ Combat · Giant Rat · 10            [Queue] [Stop]│
├──────────────────────────────────────────────────┤
│ [███████████ 120 / 350 ────────────────────────] │
│ [███████████████ 90 / 90 ──────────────────────] │
└──────────────────────────────────────────────────┘
```

- 信息区：格式同生产行动
- 控制区：Queue 按钮显示 `Queue (n)`，Stop 按钮固定显示；按钮宽度固定避免抖动
- 进度区：双条进度条，上条红色代表 HP（暗红背景 + 亮红填充 + 白色文字阴影），下条蓝色代表 MP（灰底 + 蓝色填充 + 白色文字）；条内居中显示 `当前值 / 上限`；高度统一 12px，300ms 平滑过渡
- Queue 按钮行为：点击打开 ActionQueueModal
- Stop 按钮行为：立即停止当前行动

## 整体约束

- 宽度：占满容器，最大 672px
- 行间距：10px（约 2.5rem）
- 按钮间距：6px
- 无障碍：进度条需暴露百分比信息供屏幕阅读器识别；战斗条使用 `aria-label` 区分 HP 和 MP；按钮提供明确的 `aria-label`；文字对比度 ≥ 4.5
- 错误处理：行动名称缺失时用 "Current Action" 兜底；战斗上下文丢失时自动降级为单条进度条；队列为空时隐藏 Queue 按钮
